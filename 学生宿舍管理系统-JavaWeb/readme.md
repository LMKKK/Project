## 主要功能

学生管理、宿舍楼管理、宿管人员管理、违禁记录管理、出入记录、异常报备六个模块。

用到的技术：JSP+Servlet+MySQL 

建议使用MySQL8.0及以上版本，如果不是也没有关系，更换jar包就可以
